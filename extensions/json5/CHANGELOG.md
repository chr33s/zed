# Changelog

All notable changes to this project are documented in this file.

## 0.1.1 - 2026-07-16

- Highlight unquoted object keys as JSON properties.
- Use Zed's current number and boolean highlight captures.
- Add punctuation and bracket highlighting.
- Add comment-language injection support.
- Add bracket matching, automatic indentation, document outline, Vim comment
  text objects, syntax overrides, and collaboration redactions.
- Fix line and block comment configuration and add JSON5-aware auto-closing.

## 0.1.0 - 2025-09-21

- Update tree-sitter-json5 to v0.1.0 and pin it by commit SHA.
